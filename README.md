# ICC - Itay's C Compiler
This project's mission is to build a self-hosting C compiler that will support most of C90's features.
The backend will generate x86 assembly.


## Features
### Lexer:
- [x] Identifiers
- [x] String literals
- [x] Integer constants
- [ ] Floating constants
- [x] Punctuators
- [ ] Preprocessor tokens
### Parsing
- [ ] Nothing
### Intermediate code generation
- [ ] Nothing
### Back-end
- [ ] Nothing
